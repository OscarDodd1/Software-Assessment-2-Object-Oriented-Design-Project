### Attributes
| Car         | Stat        | Higher/Lower |
| ----------- | ----------- | ------------ |
| Top speed   | (km/h)      | ↑            |
| 0-100       | (time)      | ↓            |
| Engine Size | (cc or kWh) | ↑            |
| Range       | (km)        | ↑            |
| Weight      | (km)        | ↓            |
| Payload     | (kg)        | ↑            |
| 0-100       | (time)      | ↓            |
